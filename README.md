# Welcome to PRODUCT-IVE

A single page web application made with Laravel and Vuejs to manage and view products and their categories.


# How to use	

	1. npm install

	2. npm run dev
	
	3. composer install 
	
	4. php artisan serve

## To migrate Tables into database

    php artisan migrate
